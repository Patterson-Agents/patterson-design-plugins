# Patterson design reference

## Color rationale

Navy #003767 anchors trust across dental and veterinary audiences; sky #00A8E1
carries energy and focus states; gray #58585B is the body-text workhorse.

## Spacing scale

8px base grid: 8 / 16 / 24 / 32 / 48 / 64.

## Accessibility

WCAG AA: 4.5:1 body text, 3:1 large text. Navy on white passes at all sizes;
sky on white is large-text only.
