# design.md — Patterson Companies

Design context for coding agents. Read before generating any UI.

## Tokens

| Token | Value | Role |
| --- | --- | --- |
| --pat-navy | #003767 | Primary surfaces, headers |
| --pat-sky | #00A8E1 | Accent, links, focus |
| --pat-gray | #58585B | Body text |

Accents reference `var(--pat-sky)` — never raw hex in components.

## Voice

Plain, confident. Sentence case everywhere. Never exclamation marks.
