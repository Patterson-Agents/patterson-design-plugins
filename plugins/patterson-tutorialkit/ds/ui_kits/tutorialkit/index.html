Define the Patterson brand tokens

A design system starts with tokens — named values that stand in for raw hex, pixel, and font values. Patterson's palette is carried by navy #003767 and sky #00A8E1, with a cool gray for text.

ⓘ What you'll build
A brand-tokens.css declaring the brand primitives plus one semantic alias. The terminal runs node validate.mjs to lint it.
▲ Watch out
--accent: #00A8E1; looks identical but is rejected — aliases must point at a primitive with var(--pat-sky) so one edit re-themes everything.
✓ Success looks like
Every line reads ✓ and the terminal prints PASS — on brand!
✕ If it fails
A missing primitive prints a red ✗ with the token name — add it and re-run.
✦ Tip
Stuck? Press Solve to reveal the _solution/ file, then compare.
