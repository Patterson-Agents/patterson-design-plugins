# docs/

Human-facing documentation assets.

| Path | What it is |
|---|---|
| [`index.html`](index.html) | Brand-styled overview page (catalog, install, dev environment) — the pretty version of the root README |
| [`screenshots/`](screenshots/) | One real PNG capture per plugin artifact — embedded in the catalog table and plugin READMEs |
| [`diagrams/`](diagrams/) | Hand-built brand-styled SVG diagrams (architecture, plugin anatomy, install flow, snapshot tree) |

Diagrams are plain SVG — edit with any vector tool or by hand; keep to the brand palette (navy `#003767`, sky `#00A8E1`, grays) and no emoji.

Regenerate a screenshot by opening the plugin's `ds/` entry file in a browser at ~1280px wide and capturing the top viewport.
